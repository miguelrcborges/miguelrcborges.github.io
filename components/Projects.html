simple-wm

A dynamic window manager, heavily inspired on dwm.

The goal is provide the same experience of dwm, but with better code (both in terms of performance and readability).

html2js

A simple html to javascript compiler, which would be considerably similar to frameworks that use JSX.

The goal is to provide a zero-cost abstraction that automates elements generation, which can be easily used with vanilla javascript.

This site uses html2js to generate each section.
